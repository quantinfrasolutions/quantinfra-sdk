# QuantInfra SDK

The QuantInfra SDK provides the interfaces, base classes, and APIs required to develop custom trading strategies and indicators for the QuantInfra platform.

## Documentation

- Strategy Development Guide: https://quantinfra.gitbook.io/quantinfra-docs/strategies/implementing-strategies
- Indicator Development Guide: https://quantinfra.gitbook.io/quantinfra-docs/strategies/implementing-custom-indicators
- Standard Indicators Package: https://github.com/quantinfrasolutions/quantinfra-standard-indicators

## Installation

```bash
dotnet add package QuantInfra.Sdk
```

Or via the NuGet Package Manager:

```powershell
Install-Package QuantInfra.Sdk
```
